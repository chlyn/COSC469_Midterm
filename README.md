## Item 1 – HTML Form Interface

This component implements a simple user interface that allows users to enter a **username and password** using an HTML form.

The form demonstrates basic **frontend and backend interaction** using:

- HTML
- CSS
- JavaScript
- Node.js
- Express

The system processes user input and demonstrates basic form handling and validation.

## Item 2 – Shared Library and Makefile

This component demonstrates how to create a **shared library in C** and compile it using a **Makefile**.

The library contains functions for:

- **Addition**
- **Subtraction**
- **Multiplication**
- **Division**

The Makefile automates the compilation process for building the shared library and linking it with the main program.

## Item 3 – Node Version Manager and Package.json

This component demonstrates installing and using **Node Version Manager (NVM)** to manage Node.js versions.

It also explains how **`package.json`** works in a Node.js project, including:

- Managing project dependencies
- Defining project metadata
- Running scripts
